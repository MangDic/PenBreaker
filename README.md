# 🎨 PenBreaker
- 간단한 그림 그리기 어플리케이션

---
### 기능
- 3가지 종류의 펜 모드를 지원합니다.
- 그리기 및 지우개, 되돌리기 등의 기능을 제공합니다.
- 완성한 그림을 디바이스에 저장하거나 공유를 할 수 있습니다.


---
### 사용 기술
 - RxSwift, PencilKit, SnapKit
